
# Future-Proof Campus Website

A modern, responsive landing page for campus management, built with React, Vite, and Tailwind CSS.

## Features
- Futuristic hero section and navigation bar
- Glassmorphism dashboard preview
- Responsive and accessible design
- Routing for Features, Pricing, and About Us pages

## Getting Started

### Development
1. Install dependencies:
	```
	npm install
	```
2. Start the development server:
	```
	npm run dev
	```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production
```
npm run build
```

### Preview Production Build
```
npm run preview
```

## Deployment
You can deploy the `dist` folder to any static hosting (Vercel, Netlify, Azure Static Web Apps, etc).

## Accessibility
- All navigation and buttons are keyboard accessible.
- ARIA labels and focus styles are included.
